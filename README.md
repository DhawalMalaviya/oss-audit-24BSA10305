## Project Overview

This project is part of the Open Source Software course. It includes an analysis of an open-source tool and the implementation of basic Linux shell scripts to demonstrate practical understanding.

---

## Shell Scripts

### Script 1 – System Identity Report
Displays basic system information such as kernel version, user name, uptime, and date.

### Script 2 – Package Inspector
Checks whether a package (Git) is installed and displays its version.

### Script 3 – Disk and Permission Auditor
Analyzes important system directories and shows their permissions and disk usage.

### Script 4 – Log File Analyzer
Reads a log file and counts occurrences of a keyword (like "error").

### Script 5 – Manifesto Generator
Takes user input and generates a simple open-source philosophy statement.

## How to Run

Open terminal and run:

```bash
chmod +x *.sh

./script1.sh
./script2.sh
./script3.sh
./script4.sh /var/log/syslog
./script5.sh
```
